# Analytics 페이지 디자인 확정

## 전체 구조
- **레이아웃**: 벤토 그리드 (Overview) → 카드 클릭 시 드릴다운 (Detail)
- **모바일 폴백**: 세로 스택

---

## 1. 캘린더 뷰

| 구분 | 선택 | 설명 |
|------|------|------|
| **벤토 카드** | A — 색상 강도 그리드 | 7×5 월간 캘린더, 세션 수에 따라 4단계 보라색 강도, 하단 dot 힌트 |
| **드릴다운** | 세션 목록 | 선택한 날짜의 세션 카드 나열 (프로젝트, 프롬프트, 시간, 토큰, 도구) |
| **인터랙션** | 날짜 클릭 → 드릴다운, 세션 클릭 → Daily 페이지 라우팅 |

---

## 2. 시간대 히트맵

| 구분 | 선택 | 설명 |
|------|------|------|
| **벤토 카드** | A — 클래식 매트릭스 | 7행(요일) × 24열(시간) 그리드, 녹색 4단계 강도 |
| **드릴다운** | C — 시간대별 수평 바 | 0~23시 세로 나열, 4구간 색상 (새벽/오전/오후/야간), 정확한 수치 + 구간별 합산 카드 |
| **인터랙션** | 카드 클릭 → 드릴다운, 기간 토글 (1주/4주/3개월) |

---

## 3. 비용 대시보드

| 구분 | 선택 | 설명 |
|------|------|------|
| **벤토 카드** | B — 도넛 + Breakdown | 모델별 도넛 차트 + 테이블 (토큰/비율/금액) + 캐시 적중률 바 |
| **드릴다운 메인** | A — 트렌드 차트 | KPI 카드 (총비용, 일평균, 캐시절감) + 일별 스택 바 차트 (모델별 색상) |
| **드릴다운 보조** | C — 청구서 인사이트 | input/output 분리 명세 + 최적화 제안 (선택적 표시) |
| **인터랙션** | 카드 클릭 → 드릴다운, 기간 토글 (1주/1개월/3개월) |
